## 📁 Contents

### 🔹 Problem Set 1

| # | Project | Description |
|---|---------|-------------|
| 01 | **Ripley Discounts** | Calculates discounts and total daily sales based on purchase amount |
| 02 | **Number Statistics** | Computes sum, average and count of even and odd numbers |
| 03 | **Payroll** | Calculates worker salary based on hours worked, including overtime |
| 04 | **Digital Clock** | Digital clock displaying hours, minutes, seconds and centiseconds |

### 🔹 Exercises

| # | Project | Description |
|---|---------|-------------|
| 07 | **Final Grade** | Calculates final grade using weighted scores for exams, homework and presentation |
| 13 | **Sorting** | Sorts 3 numbers from greatest to least |
| 14 | **Monthly Payment** | Calculates monthly payment with surcharge and optional advance payment |
| — | **Basic Calculator** | Performs basic arithmetic operations between two integers |

### 🔹 Problem Set 2

| # | Project | Description |
|---|---------|-------------|
| 01 | **Scholarship System** | Determines if a student qualifies for a scholarship based on GPA and career |
| 02 | **Taylor Series** | Computes cos(x) using Taylor series with a minimum of 14 terms |
| 03 | **Square Matrix** | Generates a random MxN matrix and calculates sum and average of the upper triangle |
